# Animated Galaxy Background

A stunning animated galaxy background effect rendered with HTML5 Canvas. This project features a beautifully detailed spiral galaxy with twinkling stars, cosmic dust, and dynamic particle effects.

## View Repository
[view this project on github]
(https://nirzarah.github.io/animated-background4/)

## Features

✨ **Dynamic Galaxy Rendering**
- 4-armed spiral galaxy with realistic particle distribution
- Procedurally generated star field
- Animated dust clouds and nebulae
- Orbital mechanics for galaxy particles

🌟 **Visual Effects**
- Twinkling stars with glow effects
- Sparkling particles with trails
- Animated galaxy halo
- Dust band shadows
- Color-varying stars (blue, white, golden)

📱 **Responsive Design**
- Adapts to any screen size
- Full-screen canvas covering the viewport
- Automatically resets on window resize

🎨 **Visual Customization**
- Adjustable galaxy colors and brightness
- Customizable particle counts and behavior
- Tweakable animation speeds and orbital velocities

## Files

- **index.html** - Main HTML structure
- **style.css** - Styling and animations
- **script.js** - Canvas rendering and animation logic

## How It Works

The animation is built using HTML5 Canvas with requestAnimationFrame for smooth 60fps performance:

1. **Canvas Setup** - Initializes the canvas and generates particle data
2. **Star Generation** - Creates a random star field with varied colors and brightness
3. **Galaxy Construction** - Builds a 4-armed spiral galaxy with realistic particle distribution
4. **Background Rendering** - Draws radial gradient background
5. **Particle Animation** - Animates stars, sparkles, dust, and galaxy particles
6. **Visual Effects** - Adds glows, halos, and color effects

## Getting Started

1. Open `index.html` in a web browser
2. The animation will start automatically
3. The galaxy will adapt to your screen size

## Customization

You can customize the animation by modifying values in `script.js`:

- `pointsPerArm: 2200` - Number of particles per galaxy arm
- `arms: 4` - Number of spiral arms
- `sparkles: 220` - Initial number of sparkles
- Color arrays for different element types
- Animation speeds and orbital velocities
